Friday, September 15, 2017 5:25 PM

Migrate an existing on-premises SQL Server data warehouse to Azure SQL Data Warehouse.

Replace the existing functionality of an on-premises system: setup and configure an Azure SQL Data Warehouse, validate and migrate the existing data warehouse schema and data to Azure, configure an Azure Data Factory pipeline to move the data, and configure Power BI to access the data.
